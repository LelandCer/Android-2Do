MVP:
I want to be able to create a task
I want to be able to collect tasks into lists
I want to be able to name lists
I want to be able to give lists a due date 
I want to be able to add new tasks to existing lists
I want to be able to edit tasks and list names
I want to be able to delete tasks and lists
I want to see my lists in order of due-soonest
I want to be able to mark a task as complete

Advanced:
I want to sort lists by created-at, updated-at, and name
I want to be able to hide completed lists
I want to see list progress without opening the list

Out-of-Scope:
I want to be notified when a list is about to be due
I want my lists to sync between devices
I want to see my active list on the lock screen
